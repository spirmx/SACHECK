## Quick Start

SA CHECK is a Windows desktop work board. It installs to:

```text
C:\SACHECK
```

User work files are not stored inside the app by default. Choose a Work folder from:

```text
Settings > Work Folder Source > Choose Work folder
```

Default clean Work folder:

```text
Documents\SA CHECK Work\Work
```

## 1. Add Or Create A Work Folder

- Open Settings.
- Find Work Folder Source.
- Choose Work folder.
- Select an existing Work1, Work2, team folder, or create a new folder.
- Press Sync now to scan folders and show items on the board.

## 2. Add Work Items

- Add file: copies a local file into Waiting.
- Create project: copies a whole project folder into Waiting.
- Add link: stores a URL as a shortcut.
- Create new work: creates a blank file from a configured type.
- Templates: saves reusable source files or links and creates new work from them.

## 3. Use The Board

- Waiting: new or pending work.
- Doing: active work.
- Success: completed work.
- Detail: edit name, type, status, date, note, and target path.
- Search: find by name, note, path, type, status, or date.

## 4. Calendar And Health

- Calendar tracks dates and reminders.
- Health checks missing files, duplicates, activity, snapshots, and recovery options.
- Undo can restore recent tracked edits when possible.

## 5. Install, Update, And Uninstall

- Run `SA_CHECK_Installer.exe` to install the app.
- Run a newer installer again to update the app in `C:\SACHECK`.
- Updates replace app code and bundled guide content.
- User-selected Work folders are outside the app and are not deleted by updates.
- Uninstall removes the app system files only, not Work1/Work2 or user Work folders.

## In-App Guide

The guide opens automatically for a new install or when the guide version changes. Users can close it anytime.

Users can reopen it from the question mark button in the lower-left sidebar.
