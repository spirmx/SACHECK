# SA CHECK Theme/Icon Brief

Design compact desktop-app themes for SA CHECK, an offline-first work board.

Keep the UI practical, readable, and not like a marketing landing page. Themes should include:

- App background
- Card surface
- Primary accent
- Sidebar color
- Active nav color
- Muted text color
- Border color

Icon direction:

- Friendly cartoon-style office/work icons
- Clean 1:1 square PNG
- Transparent background
- Good at 24px, 32px, and 48px
- Avoid tiny text inside icons

Sidebar media direction:

- Small 48px profile tile above online/offline status
- Works best with transparent PNG or animated GIF
- Motion should be subtle, clean, and not distracting
- Video concepts should be short seamless loops under 20 seconds, but current app build uses image/GIF runtime first

Settings layout direction:

- Split settings into clear menu/tab pages
- Theme changes should apply immediately
- Show a tiny app preview before or while selecting a theme

Current style names:

- Ocean Pro
- Candy Cartoon
- Sakura Desk
- Mint Studio
- Sunset Pop
- Night Arcade
- Graphite Focus
